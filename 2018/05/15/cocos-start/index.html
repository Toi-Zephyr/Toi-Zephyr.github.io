Cocos2d-x 安装配置

当前环境

  1. Mac 10.11.6
  2. Python 2.7.x

开工大吉

1.下载Cocos2d-x 3.16获得压缩包cocos2d-x-3.16.zip

2.解压压缩包cocos2d-x-3.16.zip

3.进入到文件夹cocos2d-x-3.16，运行

4.测试环境配置

5.完成！

HelloWorld！

新建项目

new 后面接项目名称
-p 表示包名
-l 表示开发语言
-d 表示文件存储目录

新建项目目录

运行新项目

1.打开 HelloWorld.xcodeproj

2.打开后

3.点击右上件，选择Mac平台

4.点击小三角形，运行（第一次编译有点慢）

5.完成！
