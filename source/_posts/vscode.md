# 配置你的「VS Code」

## 安装 「Power Mode」

![guoCz.png](https://s1.ax2x.com/2018/06/13/guoCz.png)

- **激情澎湃的代码！**

> Your code is powerful, unleash it! The extension made popular by Code in the Dark has finally made its way to VS Code.

相关设置：

1. 打开命令面板
2. 选择`User Setting`
3. 选择`Power Mode`

```
"powermode.enabled": true
```
